# Ziwei Doushu

`apps/ziwei/` is a clean standalone Ziwei Doushu project scaffold for `louisko.com`.

## Current Scope

- Birth input form
- 12-palace board UI
- Summary card placeholders
- Rule docs and engine module scaffolds

## Intentionally Not Finished Yet

- `命宮`
- `身宮`
- 主星安星
- 輔星安星
- 四化
- 大限 / 流年 / 流月

Until those modules are implemented, the UI intentionally renders `待補` instead of pretending the chart is complete.

## Files

- `index.html`: public app entry
- `styles.css`: visual system and responsive board layout
- `app.js`: form state and placeholder chart rendering
- `docs/`: rules and regression anchors
- `engine/src/`: future reusable Ziwei logic modules

## Local Preview

Open `apps/ziwei/index.html` in a browser or serve the repo with any static server.
