## 核心結論

- 預設排盤風格是 `三合`
- 安星與亮度判準整體明顯偏向 `中州派`
- 流年四化採 `依據流年天干`
- 閏月採 `月中分界`
- 晚子時採 `視為當日`
- 此基準可作為 Ziwei 專案第一版實作依據，但不可把「已有基準」誤當成「演算法已完成」

## 規範摘要

以下段落整合了你提供文件中的規範型敘述，方便先快速掌握方向。

### 1. 預設風格與基本語系

- 語系設定：`正體中文`
- 排盤風格：`三合（預設）`

### 2. 安星方法基準

系統在安星與星曜亮度評等上，整體採偏向 `中州派` 與傳統經典的綜合排法。

- 安天馬：依據 `年支`
- 安天空：採 `常規排法`
- 星曜亮度：依據 `中州派理論`
- 安截空旬空：採 `正副雙星法`
- 安天使天傷：採 `中州派排法`
- 安魁鉞：採 `六辛逢虎馬`
- 安命主：依據 `中州派理論`
- 流年四化：依據 `流年天干`
- 安長生十二神：
  - 區分陰陽順逆
  - 採 `水土共長生`

### 3. 三合盤顯示與界面設定

- 使用 `平鋪型` 限流按鈕面板
- 本命盤十二宮內顯示：
  - 流年虛歲年齡
  - 小限虛歲年齡
- 限流盤顯示：
  - 流年（疊宮 + 四化 + 年曜）
  - 小限（疊宮 + 四化 + 小曜）
- 中宮與標識：
  - 顯示三方四正指示線
  - 顯示非節氣四柱
  - 顯示太極點轉換標籤
  - 顯示自化箭頭圖標
  - 使用彩色多箭頭標識自化
- 星曜顯示：
  - 顯示華蓋、劫煞、咸池
  - 顯示天德、月德、龍德
  - 歲建十二神隨限流變化
  - 將前十二神隨限流變化
  - 使用黑色字標識六煞星
  - 顯示運曜和流曜
  - 顯示限流天馬、紅鸞天喜、文昌文曲

### 4. 飛星盤輔助顯示

雖然預設風格是 `三合`，但飛星盤相關的視覺輔助設定仍有開啟：

- 顯示神煞
- 顯示來因宮
- 顯示命宮四化
- 彩色多箭頭標識自化
- 中宮顯示三方四正指示線

### 5. 四化盤設定

- 顯示神煞
- 顯示流年和虛歲年齡
