年柱：甲辰

## Case 2：大運與六柱十二字同步

### 輸入

```text
1974/10/03 04:00
性別：女
流年：2026
```

### 已知頁面顯示

大運試算中，2026 年主要對應：

```text
戊辰大運
2026年03月–2036年03月
年齡：51–60歲
```

### 期望

「六柱十二字整合分析」中的大運也必須是：

```text
戊辰
```

不能顯示成：

```text
己巳
```

## Case 3：流年切換

### 操作

在會合局分析頁更改流年年份：

```text
1998
2046
```

### 期望

- 流年干支會更新
- 對應大運也會更新
- 六柱十二字統計重新計算
