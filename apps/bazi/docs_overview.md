# 系統功能總覽

本系統為完整四柱八字分析系統，包含：

- 四柱排盤
- 十神分析
- 地支本氣與藏干
- 旺衰分析
- 大運試算
- 流年查詢
- 合沖刑害分析
- 五行色彩視覺化
- A4 列印 / PDF 輸出

---

# 四柱排盤功能

## 四柱排列方式

由左至右：

1. 年柱
2. 月柱
3. 日柱
4. 時柱

每柱皆採：

- 天干在上
- 地支在下

---

## 五行顏色系統

| 五行 | 顏色 |
|---|---|
| 木 | 綠色 |
| 火 | 紅色 |
| 土 | 棕色 |
| 金 | 灰色 |
| 水 | 藍色 |

---

# 十神分析

系統以：

- 日主（日干）

作為核心判斷。

## 十神分類

| 類型 | 說明 |
|---|---|
| 同我 | 比肩、劫財 |
| 我生 | 食神、傷官 |
| 我剋 | 正財、偏財 |
| 剋我 | 正官、七殺 |
| 生我 | 正印、偏印 |

---

# 地支本氣與藏干

系統可展開：

- 主氣
- 中氣
- 餘氣

並依據日主自動判定十神。

---

# 合沖刑害分析

系統可自動判定：

- 六合
- 六沖
- 三刑
- 六害
- 半合
- 三合

並以：

- 綠色：合
- 紅色：沖 / 刑 / 害

進行視覺標示。

---

# 大運試算

## 功能內容

- 支援男命 / 女命
- 自動判定順逆行
- 自動起運
- 前後節氣起運、十步大運

---

## 大運顯示內容

每步大運包含：

- 干支
- 西元時間
- 年齡區間
- 天干十神
- 地支本氣十神
- 順逆行

---

## 大運格式

範例：

```text
丙戌
1992－2001
年齡：16－25歲
天干：丙｜劫財
地支本氣：戌｜傷官
```

目前畫面以整數年份顯示大運區間；精確交運月份與日期尚未輸出。

---

# 流年查詢

## 功能內容

可查詢：

- 指定年份流年
- 當年度干支
- 流年十神
- 地支互動

---

## 流年互動分析

系統自動分析：

- 原局 vs 流年
- 地支六合
- 地支六沖
- 三刑
- 六害

並使用標籤呈現。

---

# 旺衰分析

系統可依：

- 月令
- 節氣
- 五行生剋

進行旺衰判斷。

---
