# Smoke Test Checklist

每次修改後請至少檢查：

## A. 基本四柱校驗

預設輸入：

```text
2024/03/10 00:20
```

應輸出：

```text
年柱：甲辰
月柱：丁卯
日柱：癸酉
時柱：壬子
```
