# 版本
九步大運含年齡版

# 系統核心架構

本系統將四柱八字拆分為：

1. 時間系統
2. 干支系統
3. 五行系統
4. 十神系統
5. 地支藏干系統
6. 旺衰系統
7. 大運系統
8. 流年系統
9. 合沖刑害系統
10. 視覺輸出系統

所有模組皆以：

- 日主（日干）

作為分析核心。

---

# 一、時間系統

---

## 1.1 使用時間

輸入：

```text
西元年 / 月 / 日 / 時 / 分
```

系統轉換：

- Gregorian Date
- Julian Day Number（JDN）

---

## 1.2 時區

系統預設：

```text
UTC+8（台灣）
```

---

## 1.3 子時規則

子時：

```text
23:00～00:59
```

系統採：

```text
晚子時不換日
```

即：

- 00:20 → 當日
- 不提前算隔日

---

# 二、年柱計算

---

## 2.1 年柱依據

以：

```text
立春
```

作為歲首。

不是農曆正月初一。

---

## 2.2 年柱算法

若：

```text
出生時間 >= 當年立春
```

則：

```text
年柱 = 當年干支
```

否則：

```text
年柱 = 前一年干支
```

---

## 2.3 年干支循環

採：

```text
60甲子循環
```

---

# 四、日柱計算

---

## 4.1 日柱核心

使用：

```text
JDN（Julian Day Number）
```

計算。

---

## 4.2 基準日

系統使用：

```text
甲子日基準
```

進行：

```text
(JDN + offset) % 60
```

推算。

---

## 4.3 日柱公式

```text
index = (JDN + 偏移值) mod 60
```

取得：

- 天干
- 地支
